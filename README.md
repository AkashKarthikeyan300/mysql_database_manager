# MySQL Database Manager

Python-based command-line interface for MySQL database operations.

## Features (Completed)
- Create tables
- Insert records
- Update records with WHERE clause
- Select data (multiple modes)
- View all tables
- Drop tables
- SQL injection protection

```bash

```

Follow the interactive menu to:
1. Create database tables
2. Insert records
3. Update data
4. Query information
5. Manage tables
